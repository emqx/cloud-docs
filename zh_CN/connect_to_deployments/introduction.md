# 简介

在连接到部署之前您需要完成 [查看部署](../deployments/view_deployment.md) 所需操作，获取部署连接地址、连接端口等必要信息。

对于免费试用和共享部署 EMQ X Cloud 默认提供了 TLS/SSL 连接认证，您可能还需下载 [根证书](https://static.emqx.net/data/cn.emqx.cloud-ca.crt) 用于 TLS/SSL 连接。
